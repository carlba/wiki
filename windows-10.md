## Installation

### Macify

1. Install autohotkey
  ```
  chocolatey install -t autohotkey
  ```

2. Create a file called macify with the following contents in your home folder.
  ```
  LWin::Ctrl
  LAlt::LWin
  ```

3. Start the file
