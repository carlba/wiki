# Usage

``` yaml
example: 'this is a very very very very very very very very very very 
          very very very very very  very very very  very very very  
          very very very  very very very  very very very  very very 
          very long string'
```

``` yaml

  this is a very very very very very very very very very very 
  very very very very very  very very very  very very very  
  very very very  very very very  very very very  very very 
  very long string
```
